# Project Decisions Log

This file records every important "why" decision in the project.

## 2026-04-30 — Stack Selection
**Decision:** Next.js 15 + TypeScript + Tailwind + react-flow + Framer Motion
**Why:** AI tools work best with Next.js. react-flow handles canvas/drag-drop magic. Framer Motion is best-in-class for animations.

## 2026-04-30 — MVP Scope
**Decision:** 4 blocks only (Source, Build, Test, Deploy), 1 level
**Why:** Beginner with few hrs/week. Ship small, iterate from feedback.

## 2026-04-30 — GitHub Account Separation
**Decision:** Personal projects in ~/Documents/personal-projects/, work in ~/Documents/work-projects/
**Why:** SSH-based identity switching keeps work and personal commits cleanly separated.
